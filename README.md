# Trip-Expense-Splitter 💸
A command-line expense splitting tool built with Python. Effortlessly track shared trip expenses and calculate the minimum number of transactions needed to settle all debts! 

## 🕹️ How to Use
1. **Add an Expense**: Input the payer's name, the total cost, and the names of the members sharing the bill.
2. **Auto-Calculate**: The system instantly updates everyone's net balance behind the scenes.
3. **Settle Up**: Select the settlement option to generate a clear, step-by-step transaction guide (e.g., "A needs to pay B $500").

## ✨ Features
* **Smart Settlement Algorithm**: Uses a Greedy algorithm approach to perfectly match debtors and creditors, minimizing the total number of transactions.
* **OOP Architecture**: Clean Object-Oriented design encapsulating member states and balances.
* **Interactive CLI**: A continuous loop interface allowing users to input multiple expenses on the go.
* **Robust Parsing**: Automatically strips accidental white spaces and handles data type conversions safely to prevent crashes.

## 💻 Tech Stack
* Python 3
* Object-Oriented Programming (OOP)
* Hash Maps (Dictionaries) & List Comprehensions
